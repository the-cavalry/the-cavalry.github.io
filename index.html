The Cavalry consists of a group of open-source enthusiasts who contribute to various projects. The Light Locker project is hosted by The Cavalry on github.

Light Locker

Light Locker is a simple locker (forked from gnome-screensaver) that aims to have simple, sane, secure defaults and be well integrated with the desktop while not carrying any desktop-specific dependencies. The project is started to provide a lock screen for session switching with LightDM. It relies on lightdm for locking and unlocking your session.

Features

  • Compile-time and run-time configuration.
  • Auto starts in the desktop environment.
  • Lock the sessions when suspending the machine.
  • Timed locking. Automatically lock the session after the screen saver starts.
  • Delay VT switching (LightDM greeter) until the first user activity on a timed lock.
